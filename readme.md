 ‏אחת מנקודות הכאב של Netflix היא שתכנים יורדים מהקטלוג ואז אין גישה אליהם יותר.

אז מסתבר שיש דרך, כשאתם מורידים קובץ לצפייה באופליין הקבצים נשמרים בתיקייה ספציפית ב-Android כדי שאפליקציית Netflix תוכל לשלוף את הקבצים כשאתם ללא אינטרנט.

הצורה שבה Netflix עובדת היא מאוד פשוטה:
1. קובץ ייעודי לוידאו (nfv) מוצפן ואין מה לעשות איתו
2. קובץ לאודיו (nfa) ‏בדרך כלל גם הדיבוב העברי ואנגלי יורדים יחד

קובץ האודיו לא  מוצפן ואפשר להמיר לפורמט יותר מוכר - wav. ואז כל שנותר לעשות הוא להוריד את קובץ וידאו ממקור מוסמך😜 אהם אהם…
